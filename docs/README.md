# Documentación del proyecto

Este directorio agrupa la documentación en carpetas separadas para facilitar la navegación en el repositorio.

## Carpetas

- `docs/defensa/`
  - `DEFENSA_TECNICA.md`: defensa técnica del proyecto con enfoque iterativo.
- `docs/prompts/`
  - `PROMPTS_USADOS.md`: prompts usados y contexto de desarrollo.
- `docs/commits/`
  - `COMMITS_PROGRESIVOS.md`: lista de commits progresivos y su propósito.
- `docs/desarrollo/`
  - `DESARROLLO_POR_ETAPAS.md`: etapas de construcción y planificación.

## Archivo adicional

- `docs/LEEME_PRIMERO.txt`: instrucciones rápidas para iniciar el proyecto.
