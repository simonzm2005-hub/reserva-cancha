# Desarrollo por etapas

Este proyecto se desarrolló en etapas progresivas y documentadas con commits descriptivos.

## Etapas principales

1. **Estructura inicial del proyecto**
   - Creación del proyecto Flask.
   - Configuración de la aplicación y rutas básicas.
   - Estructura MVC inicial.

2. **Modelado de datos**
   - Agregar modelo `Cancha`.
   - Agregar modelo `Reserva`.
   - Definir atributos y validaciones básicas.

3. **Lógica de negocio**
   - Crear servicio de reservas.
   - Validar disponibilidad y horarios.
   - Calcular precios según tipo de cancha y duración.

4. **Interfaz y controladores**
   - Crear controlador de reserva.
   - Crear vistas de inicio, disponibilidad y reserva.
   - Añadir lista de reservas y cancelación.

5. **Documentación y ajustes finales**
   - Añadir documentación del proyecto.
   - Organizar archivos por carpetas.
   - Mejorar mensajes de commit.

## Mensajes de commits recomendados

- `1. Crear estructura inicial del proyecto`
- `2. Agregar modelo Cancha`
- `3. Agregar modelo Reserva`
- `4. Crear servicio de reservas`

- `6. Crear vista de inicio`
- `7. Crear formulario de reserva`
